# Figure Skating Competition Planner

En webbapplikation för att planera och schemalägga konståkningstävlingar.

## Teknisk Stack

Projektet är byggt med följande teknologier:

-   **Ramverk & Byggverktyg:** React (Vite)
-   **Språk:** TypeScript
-   **UI Komponenter:** Material UI (MUI), Emotion
-   **State Management:** Redux Toolkit, React-Redux
-   **Routing:** React Router
-   **Drag & Drop:** dnd-kit
-   **Diagram:** Recharts
-   **Validering:** Zod
-   **Övrigt:** uuid, lz-string

## Komma igång

Följ dessa steg för att köra projektet lokalt:

1.  **Installera beroenden:**
    ```bash
    npm install
    ```

2.  **Starta utvecklingsservern:**
    ```bash
    npm run dev
    ```

3.  Projektet kommer nu att vara tillgängligt på `http://localhost:5173` (eller annan port som Vite anger).

## Bygga för produktion

För att bygga applikationen för produktion:

```bash
npm run build
```

## Linting

Kör linting för att kontrollera kodkvalitet:

```bash
npm run lint
```
